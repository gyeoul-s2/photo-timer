# 📸 Photo Flow Timer

集中勉強タイマー。写真がゆっくり流れる中で時間を計ります。

**[→ デモを開く](https://gyeoul-s2.github.io/photo-timer/)**

---

## モード一覧

| モード | 説明 |
|---|---|
| ⏱ タイマー | 指定時間のカウントダウン |
| 🍅 ポモドーロ | 集中・休憩を繰り返す。セット数・時間カスタマイズ可 |
| 🕐 時刻 | 現在時刻を表示（12/24時間切替可）|
| 🔔 アラーム | 指定時刻になったら鳴る |

---

## 切替演出（10種類）

| 名前 | 説明 |
|---|---|
| なし | 即切替 |
| フェード | ふわっと |
| ディゾルブ | クロスフェード（ゆっくり重なる）|
| スライド | 横にスライド |
| ドリフト | 右から左へゆっくり流れる |
| ケン・バーンズ | ゆっくりズーム＋パン |
| ズームアウト | 引いていく |
| フリップ | カード裏返し |
| ブラー | ぼかしから現れる |
| グリッチ | ノイズで切り替わる |

## 機能

### フォント（7種類）
JetBrains Mono / Space Mono / Bebas Neue / Playfair Display / Oswald / DM Serif / Rajdhani

### タイマーカラー
- プリセット6色 / カラーホイール / HEXコード入力

### 写真オーバーレイ
- スライダーで暗さ調整（タイマーを読みやすくする幕）
- 色：黒 / 赤み / 青み / 緑み / 暖色 / 白

### 🪟 Picture in Picture (Document PiP API)
Chrome 116+ 対応。**他のタブに移動しても浮いたままで表示。**
- タイマーのみ / 写真のみ / 両方 をPiP内で切替
- PiP内の演出を別途設定可（重い端末は「なし」推奨）
- PiP起動中、元タブは軽量モードに切替（写真は薄表示、タイマー非表示）

### その他
- フォントサイズ・透明度
- タイマー位置（5箇所）
- 表示スタイル（全画面/フレーム）
- タイマー音：終了のみ / 残り10秒カウント / なし
