# 📸 Photo Flow Timer

集中勉強タイマー。写真がゆっくり流れる中で時間を計ります。

**[→ デモを開く](https://gyeoul-s2.github.io/photo-timer/)**

---

## 使い方

1. 写真をドロップまたは選択（フォルダーごとでもOK）
2. タイマー時間とモードを設定
3. スタート → マウスを動かすと下にコントロールが現れます

---

## 機能

### 写真
- ランダム順（連続重複なし）
- **演出3種**：フェード / スライド / リバー（川のように左から流れる）
- 切替間隔は自由に変更可

### タイマー
- **カウントダウンモード**：時・分・秒で設定、残り60秒/10秒で色変化
- **現在時刻モード**：タイマーの代わりに現在時刻を表示（実行中にも切替可）

### フォント（7種類）
| フォント | 雰囲気 |
|---|---|
| JetBrains Mono | コードスタイル（デフォルト）|
| Space Mono | コンパクトなモノスペース |
| Bebas Neue | 力強いディスプレイ体 |
| Playfair Display | 上品なセリフ体 |
| Oswald | すっきりした細字 |
| DM Serif Display | エレガントなセリフ |
| Rajdhani | 軽やかなジオメトリ体 |

### タイマーカラー
- プリセット6色スウォッチ
- 🎨 カラーホイール（虹色ボタンをクリック）
- HEX コード直接入力（例：`ff6b6b`）

### 写真オーバーレイ
- スライダーで暗さ 0〜88% 調整（タイマーを読みやすくする幕）
- 色：黒 / 赤み / 青み / 緑み / 暖色 / ニュートラル / 白

### 🪟 Picture in Picture（Document PiP API）
Chrome 116+ 対応。**他のタブに移動しても浮かんで表示されます。**

- タイマー数字のみ / 写真のみ / 両方　を PiP 内でトグル切替
- PiP 内から一時停止・次の写真ボタンも操作可能
- スクリーンシェアは不要 — OS ネイティブな浮きウィンドウ

### その他
- タイマーのフォントサイズ・透明度
- タイマー位置（右下/左下/右上/左上/中央）
- 表示スタイル（全画面/フレーム）
- タイマー音：終了ビープ / 残り10秒カウント / なし

## ローカルで開く

```bash
python -m http.server 8000
# → http://localhost:8000
```

`index.html` をブラウザで直接開くだけでも動作します。

---

写真はブラウザ内だけで処理されます。外部サーバーへの送信はありません。
